Snowl Update

You've just updated Snowl, an experiment in messaging in the browser.

This version includes a number of new features. In particular:

  • read messages in the stream view, a sidebar that lets you keep track of incoming messages while you do other things;
  • send messages via Twitter by pressing the write button in any view;
  • subscribe to multiple Twitter accounts;
  • add buttons to your toolbars for quicker access to Snowl features via the View > Toolbars > Customize... menu option.

It also includes many bug fixes.
