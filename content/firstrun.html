Getting Started with Snowl

Snowl is an experiment in messaging in the browser. You can access Snowl from the Tools menu or the Snowl button on the statusbar in the bottom left corner of your browser window. You can also add buttons to your toolbars for quicker access to Snowl features via the View > Toolbars > Customize... menu option.

Subscribe to feeds by clicking the feed icon in the location bar on a web site with a feed, then selecting Snowl from the list of feed readers on the feed preview page. Subscribe to Twitter or import an OPML file from the Snowl menu in the Tools menu and on the statusbar.

Read messages in the river, stream, and list views. Write a message by pressing the write button in any view.

Get involved! Learn more about Snowl, tell us what you think, browse bug reports, report a bug, and check out the source.
